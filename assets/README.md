# Assets

This directory contains static assets for the Flutter client.

## Fonts

Place `JetBrainsMono-Regular.ttf` in `fonts/` for monospace 
display of fingerprints, keys, and onion addresses.

Download from: https://www.jetbrains.com/lp/mono/

## No Other Assets

This app intentionally contains no:
- Custom icons (uses Material Icons)
- Images or graphics
- Sounds or notifications
- Branding assets

Minimalism is a feature.
